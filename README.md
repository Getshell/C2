# 《C2-远程控制的艺术》

本项目用来收集整理C2工具的相关内容，包括优秀的C2工具或优秀的思路等。远控是一门古老的技术，C2是一门新兴的艺术！不想拿到控制权限的黑客不是好黑客。

本项目创建于2021年8月19日，最近的一次更新时间为2021年8月19日。
